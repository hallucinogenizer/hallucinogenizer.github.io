IEC LMS & Acquisition System

2021-2022 NodeJS | ExpressJS | ReactJS | MySQL | AWS EC2 | AWS SES | Redis-Bull

IEC teaches emerging technologies to students. They receive thousands of applications for each cohort and use a multi-step Acquisition Process to shortlist candidates, which includes:

  1. Application
  2. Assessment
  3. Orientation
  4. Interviews
  5. Zero Week

Students that apply for IEC Cohorts often come from a low-computer-literacy background and hence had problems using the Assessments Systems IEC was using. IEC hired me to solve this problem. My Assessments System has been in use for 2 cohorts now and has had an average student rating of 4.5/5 from thousands of reviews.

The portal is server-side routed with ExpressJS and React is used merely for UI rendering. React-router was not used. This decision was made to avoid the excessive load of client-side routing on students' low-powered computers. The decision to use relational databases was made as it allows us to enforce database-level logic constraints of relationships between eitities such as Assessments, Sections, Questions, and Options. The system is hosted on AWS EC2 and uses the AWS Simple Email Service (SES).
