Recent Projects

IEC LMS & Acquisition System

 2021-2022 Full Stack Software Engineer | NodeJS | ExpressJS | ReactJS | MySQL | AWS EC2 | AWS SES | Redis-Bull

Production website hosted on AWS EC2

IEC teaches emerging technologies to students. They receive thousands of applications for each cohort and use a multi-step Acquisition Process to shortlist candidates, which includes:

  1. Application Form
  2. Online Multi Part Assessment
  3. Orientation
  4. Online Interviews
  5. Onboarding

Students that apply for IEC Cohorts often come from a low-computer-literacy background and hence had problems using the Assessments Systems IEC was using, which had been built in the US. IEC hired me to solve this problem. I started off with an Assessments system but the project later expanded to all phases of the Acquisition process.

My Acquisition System has been in use for a year now and has had an average student rating of 4.5/5 averaging thousands of student reviews. It has enabled the company to onboard a new cohort every 3 weeks, a process that previously took several months.

The portal is server-side routed with ExpressJS and React is used merely for UI rendering. React-router was not used. This decision was made to avoid the excessive load of client-side routing on students' low-powered computers. The system is hosted on AWS EC2 and uses the AWS Simple Email Service (SES).

Open Data Pakistan

2021-2022 Team Lead Software Engineer | CKAN | Python | Flask | PostgreSQL | Solr | AWS EC2

Visit Open Data Pakistan

Open Data Pakistan is run by the National Center for Big Data and Cloud Computing (NCBC), Lahore University of Management Sciences (LUMS) and Higher Education Commission (HEC). It is recognized as one of the top CKAN portals of the world by DataShades. It is a collaborative effort to help provide open data to all kinds of users.

The portal is based on the open source CKAN project, the same repository that powers other data portals such as that of the US Government and the Australian Government. It is built in Python Flask with Jinja2 templating.
